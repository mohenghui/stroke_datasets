笔画分割数据集制作   
将压缩包解压后放在文件夹中便可

# 如何运行   
python get_stoke.py    

svg2img2()为将染色/未染色的svg转换成图片生成标签,传参中label=True为染色,label=False为未染色   
get_color()为将svg按笔画类别染色
